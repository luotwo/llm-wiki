# MCP 协议

Model Context Protocol，让 AI 模型能直接连接和操控外部工具的接口协议。

## 在本知识库中的关联

[[n8n MCP Server]] 是 n8n 官方基于 MCP 协议提供的接口，让 [[Claude Code]] 能直接操作 n8n 实例——创建工作流、修改节点、运行测试，全部通过自然语言完成。

## 意义

MCP 让 AI 从"顾问"变成"操盘手"——不再只是告诉你怎么做，而是直接帮你做。
